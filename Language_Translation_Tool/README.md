# Language Translation Tool

A simple and user-friendly Language Translation Tool built using Python and Streamlit. This application allows users to translate text between multiple languages instantly.

## Features

- Translate text between multiple languages
- Easy-to-use Streamlit interface
- Fast and accurate translations
- Supports a wide range of languages
- Real-time translation results

## Technologies Used

- Python
- Streamlit
- Deep Translator

## Installation

### 1. Clone the Repository

```bash
git clone <repository-url>
cd Language_Translation_Tool
```

### 2. Install Dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the Application

```bash
streamlit run app.py
```

## How It Works

1. Enter the text you want to translate.
2. Select the source language.
3. Select the target language.
4. Click the Translate button.
5. View the translated output instantly.
